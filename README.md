# Екзамен: Валідація Email (Білет №20)

Цей проєкт містить практичну реалізацію функції для перевірки формату адреси електронної пошти та автоматизовані тести для підтвердження її коректності.

## Загальний опис
Метою роботи є створення надійного механізму примітивної перевірки (валідації) введених даних. Проєкт розділений на два модулі:
1.  **Логіка програми**: Функція, що аналізує рядок на відповідність базовим правилам (наявність символу `@` та крапки у доменній частині).
2.  **Тестування**: Набір юніт-тестів, які перевіряють роботу функції на різних типах вхідних даних (валідні адреси, адреси без спецсимволів тощо).

---

## Реалізація

Проєкт складається з двох основних файлів у папці `exam`:

* **`main.py`**: Містить функцію `validate_email(email: str)`.
* **`test.py`**: Містить клас `TestFunction` для запуску тестів через бібліотеку `unittest`.
